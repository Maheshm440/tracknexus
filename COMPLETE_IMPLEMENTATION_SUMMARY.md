## üìû SUPPORT & TROUBLESHOOTING

A: Check internet connection, or download manually from Unsplash

**Q: ImageMagick not found?**
A: Install from https://imagemagick.org/ or use TinyJPG.com online

**Q: Build fails after changes?**
A: Clear Next.js cache: `rm -rf .next` then rebuild

**Q: Images still loading from Unsplash?**
A: Run `node scripts/update-blog-data.js` again to verify all updates
